**Cairo By Example** deployed to [https://cairo-by-example.xyz](https://cairo-by-example.xyz)

## Developing

Clone the repository

```bash
git clone git@github.com:a5f9t4/cairo-by-example.git
cd cairo-by-example
```

Install Dependencies

```bash
yarn
```

Run locally

```bash
yarn dev
```
